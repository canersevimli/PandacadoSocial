Project: PandacadoSocial

Type: AI-powered social media SaaS (Turkey market)

Core Features:
- Onboarding chat
- Brand profile generation (JSON)
- 30-day content plan
- Post generation (caption + hashtags + image prompt)

Tech Stack:
- Backend: Node.js (Express)
- Frontend: React + Tailwind
- Database: MySQL

Goal:
Build a scalable SaaS for automated social media content generation.

Important:
- Keep code modular
- Avoid unnecessary complexity
- Focus on MVP first
